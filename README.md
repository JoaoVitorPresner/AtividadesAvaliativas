# AtividadesAvaliativas

Neste repositório estão presentes todas as atividades avaliativas referentes ao primeiro e segundo trimestre.

Contém os códigos e as explicações com o objetivo de:

- Organizar as atividades avaliativas

- Documentar explicações e resoluções

- Facilitar estudos e revisões futuras

## Estrutura do Repositório

📁 AtividadesPrimeiroBimestre  
→ Atividades desenvolvidas durante o primeiro trimestre

📁 AtividadesSegundoBimestre  
→ Atividades desenvolvidas durante o segundo trimestre

## Observações

Os códigos das atividades foram adicionadas em arquivos (`.c`) e as explicações em arquivos  (`.md`) para facilitar a compreensão e organização do conteúdo.
